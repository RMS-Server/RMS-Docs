## 10. 文档间跳转

### 10.1 基本语法

在 Markdown 中，可以使用相对路径或绝对路径来创建文档间的链接：

```markdown
[链接文字](路径/文件名.md)
```

### 10.2 实际示例

#### 相对路径链接
[查看项目说明](../../README.md)
[查看项目笔记](../../note.md)

#### 绝对路径链接
[查看项目信息](/projectinfo.md)

### 10.3 链接到文档特定章节

你也可以通过添加锚点链接到文档的特定章节：

```markdown
[链接到特定章节](文件名.md#章节名)
```

例如：
[查看项目技术栈](../../projectinfo.md#项目信息)
[查看项目修改记录](../../note.md#第一次修改)
