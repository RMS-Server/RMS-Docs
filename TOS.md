# Conflux Union (CXU) Minecraft Server 用户服务协议

## 关于 Conflux Union

Conflux Union / 同川世界（CXU）是一个专注于 Minecraft Java 版生电服务器。本协议适用于 CXU SSO 统一认证系统及 CXU 服务器的所有相关服务。

## 1. 协议的接受

欢迎来到 CXU 服务器！在注册账号和使用 CXU 的任何服务前，请您务必仔细阅读并充分理解本协议的全部内容。

**您一旦完成注册，即表示您已充分阅读、理解并同意接受本协议的全部内容。**

## 2. 账号注册与使用

2.1 注册要求
- 您在注册账号时应提供真实、准确的信息，包括但不限于用户名、密码和QQ号码
- 您必须是 CXU QQ群的成员才能完成注册
- 一个QQ号仅可注册一个游戏账号

2.2 账号安全
- 您应妥善保管您的账号和密码，因保管不善导致的账号被盗用等后果由您自行承担
- 您不得将账号借给他人使用，不得出售、转让账号
- 您不得恶意注册多个账号

2.3 账号责任
- 您需要对自己账号下的所有行为负责
- 如发现账号被盗用，请立即联系管理员

## 3. 用户行为规范

**您在使用 CXU 服务及在服务器内游玩时，必须严格遵守中华人民共和国相关法律法规**，包括但不限于：
- 不得发布违法违规信息
- 不得从事任何违法犯罪活动
- 不得侵犯他人合法权益

### 3.2 公序良俗

**您不得违反公序良俗**，包括但不限于：
- 不得发布、传播低俗、色情、暴力等不良信息
- 不得使用侮辱性、歧视性的用户名或聊天内容
- 不得进行骚扰、恶意攻击他人的行为

### 3.3 社区规范

**您不得发表引战言论**，包括但不限于：
- 不得挑起、参与地域歧视、人身攻击等引战行为
- 不得在公共频道进行无意义的争吵或辱骂
- 应保持理性、友善的交流态度
- 尊重其他玩家和管理团队

**您必须遵守 CXU 服务器规则**：

- 服务器规则会不定期更新，请及时关注

### 3.5 技术规范

- 不得使用作弊、外挂等第三方非法程序
- 不得利用游戏漏洞（Bug）进行恶意破坏
- 不得对服务器进行 DDoS 攻击或其他网络攻击
