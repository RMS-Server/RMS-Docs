# RMS帮助文档
_本页面意在整合公告中出现的帮助文档_

## Q&A

### 我看不懂文档/我找不到我要的内容
- 可以点击屏幕右下角的🤖标志，询问文档AI直接得到答案。
- 如果文档AI也无法给出答案。欢迎提出建议，我们将根据你的建议进一步完善文档。

### 我无法连接至服务器？

- 请检查你的网络连接是否正常
- 请检查你的游戏版本，由于服务端为1.17.1，高版本(>1.21)Via存在问题，请降低版本重试
- 由于主域名采用**SRV解析**，部分DNS可能无法解析，请尝试使用`v6.rms.net.cn:5555`(请确保你的网络环境支持ipv6)或`backup.rms.net.cn:5555`连接
- 依然存在问题请联系管理或者在[这里](https://feedback.rms.net.cn)反馈问题

### 提示未在白名单内

- 请确定已经申请了服务器白名单并且已经通过审核（添加白名单[点我](https://wl.rms.net.cn)）
- 依然存在问题请联系管理或者在[这里](https://feedback.rms.net.cn)反馈问题

### 我无法切换到其他子服？

- 请先确认其他子服是否启动。若没有，可联系管理。
- 请确定你的游戏版本，只建议使用1.17.1客户端连接服务器。若客户端版本过高(>1.21)且客户端模组极少，可能存在可以连接到服务器，但切换子服就报错的情况。

### 我连不上建筑服

- 建筑服版本为1.21.4。请确认你的客户端版本与服务器相符。
- 建筑服实体机会自动睡眠以降低功耗。可以使用以下方式唤醒。
   - ~~网页唤醒~~
   - ~~qqbot唤醒~~
   - 联系管理
- 如果唤醒后等待几分钟仍无法连接，在检查网络连接后可联系管理

### 我遇到了bug/我有个建议

- 我们欢迎玩家为RMS这个大家庭建言献策。你可以通过以下途径为RMS添砖加瓦：[RMS反馈系统](https://feedback.rms.net.cn)
- 如果因为意外情况无法访问[RMS反馈系统](https://feedback.rms.net.cn)，可以直接向管理反馈情况。

# 关于迁移小地图路径点的方法
_该方法仅适用于Xaero地图_
> 假设原服务器为`example.com`，新服务器为`game.rms.net.cn`

1. 首先打开你的 `.minecraft` 文件夹

2. 寻找原路径点：
   - 找到原路径点文件夹：`.minecraft\XaeroWaypoints\Multiplayer_example.com`
   - 复制该文件夹中的所有文件

3. 找到新路径点：
   - 找到新路径点文件夹：`.minecraft\XaeroWaypoints\Multiplayer_game.rms.net.cn`
   - 删除该文件夹中的所有文件
   - 将之前复制的路径点文件粘贴进去

4. 寻找原地图文件：
   - 找到原地图文件夹：`.minecraft\XaeroWorldMap\Multiplayer_example.com`
   - 复制该文件夹中的所有文件

5. 找到新地图文件：
   - 找到新地图文件夹：`.minecraft\XaeroWorldMap\Multiplayer_game.rms.net.cn`
   - 删除该文件夹中的所有文件
   - 将之前复制的地图文件粘贴进去
