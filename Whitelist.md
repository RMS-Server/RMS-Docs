# RMS白名单

> RMS白名单系统采用现代化的网页管理模式，绝大部分操作可由玩家自主完成而并非需要管理员的参与

如果在使用过程中遇到问题，请联系管理层成员

## 对于新玩家
2. 打开浏览器，访问https://wl.rms.net.cn
3. 点击`通过SSO登录申请`
4. 注册账号（用户名要为你的游戏ID）
5. 等待跳转回白名单网页
6. 检查你的ID是否存在于白名单中

## 关于改名
1. 登录[RMS SSO 用户中心](https://sso.rms.net.cn/users)
2. 点击`修改用户名`，修改为你的新的游戏ID
3. 打开[RMS白名单系统](https://wl.rms.net.cn)，检查自己的新ID是否在白名单内
