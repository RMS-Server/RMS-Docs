# 特性及功能
## 开启的Carpet特性
- 烧车
- carpet自动合成
  - 1.21更新后，自动合成受到很多玩家的喜爱。在`Logic_VY`的提议下，经管理层讨论，在多方面权衡之后，决定开启carpet的投掷器自动合成，与原版“牛爷爷”的逻辑有所区别。
  
  - 具体逻辑为：如果一个指向工作台的投掷器含有一个合法的3x3合成配方时被激活，会按照这个配方进行合成。令比较器检测这类投掷器时检测非空格子数量，令漏斗、投掷器、发射器向这类投掷器输入时只能输入1物品/格 (不影响玩家向投掷器内输入)。该特性开启后，将对我们服务器的收集系统产生巨大的改变，探索研究空间充裕。望各位玩家多多研究，丰富此科技树。
- 假人背包
- 鞘翅再生
  - 基于TIS Carpet Addition 的 `renewableElytra`, 即幻翼被潜影贝杀死有几率掉落鞘翅

## RMS移植的高版本特性
- 含水树叶
  - 和高版本原版的含水树叶没有任何区别。在RMS中，含水树叶会有水滴的粒子效果和在树叶表面冒出气泡作为辨别手段

## MCDR
基本结构: 同一个插件几乎都已!!xxx 起始；很多都可以通过!!xxx 或者 !!xxx help 查看常用信息
### location list
- !!loc list 显示所有路标
- !!loc add <名称> here
- !!loc add <名称> x y z
### @@
- @ <玩家ID> 对应玩家播放一个提示音
- @@ <玩家ID> 对应玩家播放很多提示音，并闪烁标题 XXX在@你

***注意有管理在的时候少用***
