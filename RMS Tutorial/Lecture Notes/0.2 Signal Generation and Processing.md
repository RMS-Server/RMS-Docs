# 原件基础：红石信号的生成和处理
1. 红石粉
   2. 火把
      1. 附着在一个方块的侧面或者上面
      2. 激活除附着方块意外毗邻的原件
      3. 充能正上方的方块
   3. 比较器
      1. 直接输出其输入的信号，强度保持不变
      2. 比较模式
      3. 减法模式
   4. “输入原件”：信号的源头
      1. 红石信号类
         1. 按钮
         2. 踏板
         3. 绊线
         4. 拉杆
         5. 阳光传感器
      2. 比较器类
         1. 检测容器容量
            1. 箱子、木桶、投掷器、发射器、漏斗等
            2. 堆肥桶、炼药锅
            3. 蛋糕、唱片机
         2. 容器屏蔽
            1. 当一个比较器既接受到来自检测容器的信号，又接受到红石能量，此时比较器会优先输出容器对应的信号，只有当红石能量强度为15时，才会输出红石能量（即直接输出强度为15的信号）
