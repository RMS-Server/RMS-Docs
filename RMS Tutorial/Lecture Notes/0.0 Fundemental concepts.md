# 单位与基本概念
1. 时间单位
   1. 现实时间单位：s
   2. 游戏最小时间单位：gt(gametick)
      1. 1s=20gt
      2. 因服务端卡顿或外部因素(如carpet)改变时，可能不满足上述等号
2. 长度单位
   1. 不同尺度下的距离单位
      1. 像素(16x)
      2. 一格(一米)
      3. 区块
   2. 不同的距离度量方式
      1. 欧几里得距离
      2. 曼哈顿距离
      3. 切比雪夫距离
3. 数量单位
   1. 个
   2. 组
   3. 盒
   4. 箱盒
4. 实体与方块
